# Grocery AI 🥦

Grocery AI is a modern RAG (Retrieval-Augmented Generation) application designed as a nutrition knowledge assistant. It allows users to ask questions about nutrition, food storage, protein content, and grocery products, receiving informed, cited answers powered by LLMs.

This project was built as part of a hands-on learning roadmap to master the foundational concepts of building a real-world RAG system from scratch.

## Purpose
The primary goal of Grocery AI is to bridge the gap between theoretical understanding of NLP/RAG concepts and practical application. We focus on building core components—such as chunking, vector mathematics, and retrieval—manually before comparing them with production-grade libraries.

## Key Features
- **Intelligent Assistant:** Answers nutrition-related queries based on provided context.
- **Source Citations:** Provides transparency by displaying the best source for each answer.
- **Modern UI:** Responsive, dark-themed chat interface built with React, TypeScript, and Tailwind CSS.
- **Robust Backend:** High-performance API powered by FastAPI.
- **Learning-First Design:** Core RAG pipeline (retrieval, prompt building, etc.) implemented to understand the mechanics behind the "magic."

## Tech Stack
- **Backend:** Python, FastAPI, Pydantic, `google-genai` (Gemini API)
- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS
- **Database:** Vector-based JSON storage
- **Core Concepts:** RAG (Retrieval-Augmented Generation), Vector Mathematics, NLP foundations

## Setup

### 1. Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a Python virtual environment (recommended).
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up environment variables:
   - Create a `.env` file in the `backend/` directory.
   - Add your Gemini API key:
     ```text
     API_KEY=your_gemini_api_key_here
     ```
5. Run the backend server:
   ```bash
   uvicorn api.app:app --reload
   ```

### 2. Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## API Documentation
The API documentation is automatically generated by FastAPI. Once the backend is running, navigate to `http://127.0.0.1:8000/docs` to explore the endpoints.
